## Installation

```bash
$ yarn install
```

## Running the app

```bash
# development
yarn start

# watch mode
yarn start:dev

# production mode
$ yarn start:prod
```

## Test

```bash
# unit tests
$ yarn test

# e2e tests
$ yarn test:e2e

# test coverage
$ yarn test:cov
```

## Lints

```bash
$ yarn format:check

$ yarn lint
```

## Generate

```bash
$ yarn generate module with-crud --name myModule
```
